## Run with Docker

```console
docker pull ghcr.io/petrisim/petri-sim-tool:latest
```

Starting a container:

```console
docker run -p 3000:3000 -d ghcr.io/petrisim/petri-sim-tool:latest
```

## Run from source

1. Install node

2. Install npm 

3. Clone git repository


4. Go to the project directory

```console
cd petri-sim-tool 
```

5. run: 

```console
npm install
```

6. run:
```console
npm start
```
